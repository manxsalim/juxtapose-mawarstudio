Juxtapose Embedder Example

This page demonstrates how to use the embedder.js script to embed juxtapose sliders on any website.

Basic Example

This is the simplest way to embed a juxtapose slider:

<div class="juxtapose-embed" 
     data-before="before.jpg" 
     data-after="after.jpg"
     data-before-label="Before" 
     data-after-label="After"></div>

Custom Size Example

You can customize the size of your slider:

<div class="juxtapose-embed" 
     data-before="before.jpg" 
     data-after="after.jpg"
     data-before-label="Original" 
     data-after-label="Modified"
     data-width="800px"
     data-height="450px"></div>

Full Remote Example

You can also use images hosted elsewhere:

<div class="juxtapose-embed" 
     data-before="https://example.com/path/to/before.jpg" 
     data-after="https://example.com/path/to/after.jpg"
     data-before-label="Before" 
     data-after-label="After"></div>

Note: For this to work, you need to replace the URLs with valid image URLs.
